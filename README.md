# BA Editor 🚀

A modern, fast, and lightweight text editor tailored specifically for Business Analysts (BAs), Systems Analysts, and Product Managers. 

This project follows an open-client model: the entire frontend is completely open-source for the community, while advanced cloud features and integrations run securely on the backend.

## 🔥 Key Features

* **Built for Analysts by Analysts:** An interface optimized for rapid documentation of requirements, Use Cases, User Stories, and traceability.
* **Markdown-First:** A seamless writing experience powered by TipTap, featuring full hotkey support for lightning-fast workflows.
* **Clean UI/UX:** Minimalist, distraction-free design built with Tailwind CSS v4 and DaisyUI.
* **Localization:** Out-of-the-box multilingual support powered by `svelte-i18n`.

## 🛠 Tech Stack

* **Frontend:** SvelteKit (v5) + Vite + TypeScript
* **Styling:** Tailwind CSS v4 + DaisyUI
* **Editor Core:** TipTap Editor
* **Backend Ecosystem:** Supabase (Auth, DB, RLS, SSR)
